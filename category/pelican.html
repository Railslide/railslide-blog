Up and running!

mar 03 giugno 2014
By Giulia Vergottini

In Pelican.

tags: pelicanpublishinggithub pages

Ok, my Pelican and GitHub pages powered blog seems to be up and running! Yay!

For setting it up I started by following this posts by Amy Hanlon. All went great until I decided to take a look at the code of the makefile and discovered a make github option. Thus I started wondering whether there was a better way to integrate Pelican with GitHub pages and landed on a post by Martin Brochhaus, which convinced me to use project pages rather than personal ones.

The only thing that remained a bit unclear in my journey toward my Pelican powered blog was when I was supposed to push the source code. Here's the workflow I eventually came up with:

# Write some content

# check if everything works fine
make devserver

# pushing the source repo
git push origin master

# push the output folder to GitHub pages
make github

# stop the dev server
./develop_server.sh stop

# celebrate!
